# DQN_CartPole
Solves the OpenAI gym 'CartPole-v0' problem using DQN.  


## Requirements
- python==3.9
- gym==0.15.4
- numpy==1.23.5
- pytorch==1.31.1  
- matplotlib



  
## Run
```
python3 main.py 
```


This code is based on [Playing Atari with Deep Reinforcement Learning](https://arxiv.org/abs/1312.5602)

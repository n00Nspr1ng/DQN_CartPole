# DQN_CartPole
This package solves the OpenAI gym 'CartPole-v0' problem using DQN.  


## Requirements
- python==3.9
- gym==0.15.4
- numpy==1.23.5
- pytorch==1.31.1  




  
## Run
```
python cart_pole_example.py 
```


This code is based on [Playing Atari with Deep Reinforcement Learning](https://arxiv.org/abs/1312.5602)
